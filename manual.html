Manual Técnico del Sitio Web


Introducción

El presente proyecto consiste en la creación de un sitio web utilizando HTML, CSS y JavaScript en Visual Studio Code.

El sitio fue desarrollado utilizando marcos (frames), permitiendo mostrar un menú lateral y diferentes páginas dinámicas.


Herramientas Utilizadas

  • Visual Studio Code
  • HTML5
  • CSS3
  • JavaScript
  • Live Server

Funciones utilizadas en Visual Studio Code

Función Uso
Open Folder Abrir la carpeta del proyecto
New File Crear archivos HTML, CSS y JS
Live Server Ejecutar el sitio web
CTRL + S Guardar cambios
F5 Actualizar el navegador

Etiquetas HTML Utilizadas

DOCTYPE

<!DOCTYPE html>

Define que el documento utiliza HTML5.

html

<html lang="es">

Etiqueta principal del documento.

head

<head>

Contiene configuraciones del sitio web.

title

<title>Mi Sitio Web</title>

Cambia el nombre de la pestaña del navegador.

body

<body>

Contiene el contenido visible del sitio.


Uso de Marcos

frameset

<frameset cols="25%,75%">

Divide la pantalla en dos marcos: uno para el menú y otro para el contenido.

frame

<frame src="menu.html">

Carga páginas dentro de cada marco.


Enlaces del Menú

<a href="notas.html" target="contenidoFrame">

Permite abrir páginas dentro del marco principal.


Propiedades CSS Utilizadas

background

background: #0f172a;

Cambia el color de fondo.

linear-gradient

background: linear-gradient(135deg,#020617,#1e293b);

Crea fondos degradados modernos.

color

color: white;

Cambia el color del texto.

border

border: 2px solid #00ffff;

Agrega bordes alrededor de elementos.

border-radius

border-radius: 15px;

Redondea las esquinas.

box-shadow

box-shadow: 0px 0px 20px #00ffff;

Agrega sombras y efectos glow neón.

padding

padding: 20px;

Agrega espacio interno.

margin

margin: 10px;

Agrega espacio externo.

transition

transition: 0.3s;

Crea animaciones suaves.

transform

transform: scale(1.05);

Permite agrandar elementos.

display: flex

display: flex;

Organiza elementos horizontalmente.

flex-wrap

flex-wrap: wrap;

Permite bajar elementos a otra línea.

object-fit

object-fit: contain;

Hace que las imágenes se adapten sin recortarse.


Uso de iframe

<iframe src="https://www.youtube.com/embed/">

Permite mostrar videos de YouTube dentro del sitio.

Conclusión

El proyecto permitió aplicar conocimientos de HTML, CSS y JavaScript para crear un sitio web dinámico, moderno e interactivo.

Además, se utilizó Visual Studio Code como entorno de desarrollo principal, facilitando la edición y prueba del proyecto mediante Live Server.
